Documentation of user_interface.py¶

This module implements FLRW-Net’s UI.

user_interface.centering_window(window, width, height)[source]¶

Center a specified window on the screen.

user_interface.config_training_window(window, output_widget)[source]¶

Set up the training window.

user_interface.create_boundary_menu(menubar)[source]¶

Create a menu to choose the type of boundary data used: edge lengths or scale factors.

user_interface.create_help_menu(root_window, menubar)[source]¶

Create a menu containing misc features that inform and help the user.

user_interface.create_threshold_menu(menubar)[source]¶

Create a menu to allow the user to choose different thresholds for the loss.

user_interface.create_triangulation_menu(menubar)[source]¶

Create a menu to choose the triangulation of the spatial hypersurfaces.

user_interface.create_weights_menu(root_window, menubar)[source]¶

Create a menu to allow the user to choose custom/pre-trained weights, which the network uses initially as its trainable weights.

user_interface.default_weights_chosen(new_value)[source]¶

Default the trainable weights initially used by the network.

user_interface.git_message_box(root)[source]¶

Show a message box containing the link the the Git project.

user_interface.load_weights(window, fname, new_value)[source]¶

Load custom initial trainable weights for the network from a specified file.

user_interface.load_weights_dialog(window, value)[source]¶

Set up load weights dialog window.

user_interface.loading(loading_window, root_window, progress_bar, i=0)[source]¶

Simulate a loading at startup with a ttk progress bar.

user_interface.my_copyright()[source]¶

Show the copyright statement.

user_interface.on_closing_root_window(root_window)[source]¶

Set closing dialog.

user_interface.open_documentation()[source]¶

Open the documentation in the user’s default webbrowser.

user_interface.open_link(window)[source]¶

Open the link in a web browser.

user_interface.purpose()[source]¶

Show the purpose of FLRW-Net upon clicking on the according button.

user_interface.run_user_interface()[source]¶

Specify the UI’s windows and features.

user_interface.save_graph(training_window, dialog, fname, fig)[source]¶

Save the graph containing the network’s training behaviour to a .png file.

user_interface.save_graph_dialog(window, fig)[source]¶

Show saving graph window upon clicking the according button in the training window.

user_interface.save_output(training_window, dialog, fname, output)[source]¶

Save the network’s output to a .json file.

user_interface.save_output_dialog(window, output)[source]¶

Show saving output window upon clicking the according button in the training window.

user_interface.save_weights(training_window, dialog, fname, weights)[source]¶

Save the trained weights of the network to a .json file.

user_interface.save_weights_dialog(window, np_trained_weights)[source]¶

Show saving weights window upon clicking the according button in the training window.

user_interface.set_default_weights()[source]¶

Set the network’s initial trainable weights to their default value.

user_interface.set_threshold(string)[source]¶

Set the value for the global loss threshold.

user_interface.startup_loading_window(root_window)[source]¶

Set up the startup loading window.

user_interface.update_button_command(root, button, time_steps)[source]¶
user_interface.update_flrw_config(input_type, value)[source]¶
