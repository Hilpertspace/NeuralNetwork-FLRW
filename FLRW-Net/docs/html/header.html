Documentation of header.py¶

This module stores general functions that fit in neither category.

header.compute_lambda_limit(nte)[source]¶

Compute an upper boundary related to the maximum value of the cosmological constant for which there is a solution given the prescribed boundary data.

header.compute_spatial_edges(sf, n3)[source]¶

Compute the spatial edge lengths if scale factors (sf) are prescribed as boundary data.

header.compute_value(n1, n3, lamb, ini, fin)[source]¶

Compute a value related to the cosmological constant to determine whether it lies above the maximum value for which there is a solution.

header.handle_warning_messages(triangulation_warning, boundary_warning, triangulation, compute_button, entry1, entry4, entry5, entry6, n1, n3, nte, ini, fin)[source]¶

Implementation of the warning messages for the different cases.

header.redirect_output_to_widget(widget)[source]¶

Switch the standard output to the scrolled text widget in the training window. Standard output and error prompts are redirected.

header.set_boundary_params(boundary, entry2, entry3, n3)[source]¶

Set the parameters that specify the boundary data.

header.set_triangulation_params(triangulation)[source]¶

Set the parameters that specify the spatial triangulations.
