Documentation of OneStep/neural_network.py¶

This module implements FLRW-Net.

class OneStep.neural_network.Network(*args, **kwargs)[source]¶

Bases: Model

Definition of FLRW-Net. The network takes the boundary edges and solves the EOMm for the struts, which it returns.

abort()[source]¶

Abort the training if the according button is clicked in the training window.

call(inputs)[source]¶

Forward-feed when the model is used.

crop(arg)[source]¶

Set the argument range of ‘tf.acos’ to [-1., 1.] since numeric computations could lead to values that are slightly outside this range due to computational error.

custom_compute_loss(prediction)[source]¶

Compute the network’s loss by evaluating the EOMm term, which should be 0 in the presence of a classical solution. Thus, its value represents a natural choice for the loss.

custom_train_step(inputs)[source]¶

Define a single step of training. ‘@tf.function’ speeds up training incredibly: — DO NOT REMOVE! —

get_min_weights()[source]¶

Output the trainable weights of the hidden layer that produced the minimum loss.

initialize_params(n1, n3, nte, lamb, loss_threshold, training_window)[source]¶

Initialize training constants.

set_weights(weights)[source]¶

Get trainable weights of a pre-trained model and hand it to the hidden layer.

training(inputs, epochs)[source]¶

Training logic of the neural network. The train_step function is called iteratively until the specified number of epochs is reached.
